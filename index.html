Are We There Yeti?

'Are We There Yeti?' is an obstacle/runner game created for the Kinect, made together with 4 other students for the Game Projects 2 course.

The goal of the game is to reach the end of the level within a certain time limit, while dodging and jumping over obstacles and collecting energy orbs.

The game is created with Unity and also playable with the keyboard.

I was on the team as a programmer and worked mainly on the pickups, the energy system and the bird companion.

Inception Rush

'Inception Rush' is a game created with 4 other students for the Game Projects 1 course. The game had to be single player, Xbox controller compatible and made using MonoGame and Microsoft XNA, with ‘rush’ as the given theme.

The main goal is to get as far as possible, by thinking fast and acting quickly.

I was on the team as a programmer and worked on: level mechanics, enemies, particles, HUD and sound.

Software used: Microsoft Visual Studio.

Level Scripting

This is a game created for the Level Scripting course, where we had to make a game from scratch using the blueprints visual scripting system of Unreal Engine 4.

Everything seen in the game is done by me. Models and animations are made in Autodesk 3ds Max, everything else is done in Unreal Engine 4.
